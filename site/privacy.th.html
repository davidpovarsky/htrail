นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด: 21 มิถุนายน 2026

ฉบับย่อ HTTrail ทำงานทั้งหมดบนอุปกรณ์ของคุณเอง ไม่มีเซิร์ฟเวอร์ ไม่มีการวิเคราะห์ข้อมูล ไม่มีบัญชีผู้ใช้ และไม่มีการเก็บข้อมูลการใช้งาน ทราฟฟิกเครือข่ายที่ดักจับได้ — รวมถึงข้อมูลที่ละเอียดอ่อนที่สุดของคุณ — จะถูกจัดเก็บไว้บนอุปกรณ์ของคุณเท่านั้น และเราไม่มีวิธีการทางเทคนิคใด ๆ ที่จะรับ มองเห็น หรือเก็บรวบรวมข้อมูลนั้นได้

1. ขอบเขตและเราเป็นใคร

นโยบายความเป็นส่วนตัวนี้อธิบายว่าแอปพลิเคชัน HTTrail ("HTTrail" หรือ "แอป") และเว็บไซต์นี้จัดการข้อมูลอย่างไร HTTrail ให้บริการโดย Anusoft ("เรา" หรือ "ของเรา") นโยบายนี้ใช้บังคับกับแอปเวอร์ชัน macOS และ iOS รวมถึงเว็บไซต์ HTTrail

HTTrail เป็นเครื่องมือสำหรับนักพัฒนาที่ทำหน้าที่เป็นพร็อกซีดักจับ HTTPS และไคลเอนต์ API ออกแบบมาให้คุณยังคงเป็นผู้ครอบครองทุกสิ่งที่มันสัมผัสแต่เพียงผู้เดียว นโยบายนี้อธิบายการออกแบบดังกล่าวอย่างละเอียดเพื่อให้คุณสามารถตรวจสอบได้

2. สิ่งที่เราไม่เก็บรวบรวม

เราต้องการให้ชัดเจนไม่มีข้อกำกวม แอปนี้ ไม่:

  • เก็บรวบรวม ส่ง หรือจัดเก็บข้อมูลส่วนบุคคลใด ๆ บนเซิร์ฟเวอร์ที่เราดำเนินการ — เราไม่ได้ดำเนินการเซิร์ฟเวอร์ดังกล่าวเลย;
  • มีการวิเคราะห์ข้อมูล การติดตาม การโฆษณา การรายงานข้อขัดข้อง หรือข้อมูลการใช้งานในรูปแบบใด ๆ;
  • ต้องการบัญชี การเข้าสู่ระบบ การลงทะเบียน หรือที่อยู่อีเมลในการทำงาน;
  • มี SDK ของบุคคลที่สามที่ฝังอยู่ซึ่งจัดทำโปรไฟล์ของคุณหรือรายงานกิจกรรมของคุณ;
  • ขาย ให้เช่า หรือแบ่งปันข้อมูลใด ๆ กับใคร — เพราะเราไม่เคยได้รับข้อมูลใด ๆ เลย

ไม่มีการ "โทรกลับบ้าน" (phone home) แอปทำงานได้อย่างสมบูรณ์แบบออฟไลน์ ยกเว้นคำขอเครือข่ายที่ คุณ สั่งให้มันทำ (ดู ส่วนที่ 5)

3. ข้อมูลที่ HTTrail ประมวลผลบนอุปกรณ์ของคุณ

เพื่อทำหน้าที่ของมัน HTTrail ประมวลผลทราฟฟิกเครือข่ายที่คุณเลือกให้ส่งผ่านมัน เมื่อคุณเปิดใช้งานการดักจับสำหรับโฮสต์หนึ่ง ๆ ข้อมูลนี้อาจรวมถึงเนื้อหาที่ละเอียดอ่อนอย่างยิ่ง เช่น:

  • URL วิธีการ (method) ส่วนหัว (header) และคุกกี้ของคำขอและการตอบกลับ;
  • โทเค็นการยืนยันตัวตน คีย์ API และข้อมูลรับรองเซสชัน;
  • เนื้อหา (body) ของคำขอและการตอบกลับ (เช่น JSON ข้อมูลฟอร์ม HTML และรูปภาพ);
  • เวลา รหัสสถานะ และข้อมูลเมตาของการเชื่อมต่อ

ทั้งหมดนี้ถูกประมวลผลและจัดเก็บไว้ บนเครื่อง ทราฟฟิกที่ดักจับได้ ประวัติคำขอ คอลเลกชัน สภาพแวดล้อม และการตั้งค่าที่บันทึกไว้ จะถูกเขียนลงในที่จัดเก็บข้อมูลของอุปกรณ์ของคุณเอง — ไดเรกทอรี Application Support ของผู้ใช้บน macOS และคอนเทนเนอร์ของแอป (หรือคอนเทนเนอร์ App Group ที่ใช้ร่วมกัน) บน iOS ไม่มีข้อมูลใดถูกส่งมายังเราหรือไปยังบุคคลที่สามใด ๆ โดยแอป

เนื่องจากข้อมูลนี้อยู่บนอุปกรณ์ของคุณ คุณจึงเป็นผู้ควบคุมมัน: คุณสามารถลบทราฟฟิกที่ดักจับได้ ล้างประวัติ หรือลบแอปและข้อมูลของมันได้ทุกเมื่อโดยใช้เครื่องมือปกติของระบบปฏิบัติการของคุณ

4. ผู้ออกใบรับรองและกุญแจของคุณ

เพื่อถอดรหัสทราฟฟิก TLS (HTTPS) HTTrail สร้าง ผู้ออกใบรับรอง (CA) รากของตัวเองขึ้นบนอุปกรณ์ของคุณ และออกใบรับรองปลายทาง (leaf certificate) ต่อโฮสต์แบบอายุสั้นที่ลงนามโดยผู้ออกใบรับรองนั้น กุญแจส่วนตัวของ CA ถูกสร้างขึ้นบนอุปกรณ์ของคุณและ ไม่เคยออกจากมันเลย เราไม่เคยมองเห็น ไม่เคยรับ และไม่มีสำเนาของมัน

หมายเหตุด้านความปลอดภัยที่สำคัญ การติดตั้งและไว้วางใจ CA รากของ HTTrail บนอุปกรณ์ จะอนุญาตให้ซอฟต์แวร์บนอุปกรณ์นั้น — รวมถึง HTTrail — ถอดรหัสทราฟฟิก HTTPS ของอุปกรณ์นั้นได้ ติดตั้ง CA เฉพาะบนอุปกรณ์ที่คุณเป็นเจ้าของหรือควบคุมเท่านั้น ปกป้องกุญแจส่วนตัวของมันให้ดี และนำ CA ออกเมื่อคุณไม่ต้องการดักจับทราฟฟิกอีกต่อไป ดูรายละเอียดเพิ่มเติมได้ที่ ข้อกำหนดการใช้งาน

5. การเชื่อมต่อเครือข่ายที่ HTTrail ทำขึ้น

HTTrail ทำการเชื่อมต่อเครือข่ายขาออกเฉพาะเมื่อเป็นผลโดยตรงจากการกระทำของคุณเท่านั้น:

  • การทำพร็อกซี เมื่อคุณส่งทราฟฟิกผ่านพร็อกซี HTTrail จะส่งต่อแต่ละคำขอไปยังเซิร์ฟเวอร์ต้นทางที่ คุณ กำลังติดต่ออยู่แล้ว (เช่น API ที่คุณกำลังทดสอบ) ข้อมูลนั้นจะไปยังบุคคลที่สามนั้นโดยตรง ไม่ใช่มายังเรา
  • การเล่นซ้ำและการเขียนคำขอ เมื่อคุณส่งคำขอจากตัวเขียนคำขอหรือเล่นซ้ำทราฟฟิกที่ดักจับได้ HTTrail จะติดต่อปลายทางที่คุณระบุ

ข้อมูลใด ๆ ที่คุณส่งไปยังเซิร์ฟเวอร์ของบุคคลที่สามด้วยวิธีนี้ จะอยู่ภายใต้แนวปฏิบัติด้านความเป็นส่วนตัวของเซิร์ฟเวอร์ นั้น ไม่ใช่ของเรา HTTrail คือผู้ส่งสารที่คุณเป็นผู้ควบคุม มันไม่ได้เพิ่มเราเข้าไปเป็นผู้รับ

6. เว็บไซต์นี้

เว็บไซต์นี้เป็นชุดของหน้าเว็บแบบสแตติกที่โฮสต์อยู่บน GitHub Pages ไม่มีการตั้งคุกกี้ ไม่มีการวิเคราะห์ข้อมูล และไม่โหลดฟอนต์ สคริปต์ หรือตัวติดตามของบุคคลที่สาม เช่นเดียวกับเว็บไซต์ใด ๆ ผู้ให้บริการโฮสต์ (GitHub, Inc.) อาจบันทึกข้อมูลทางเทคนิคมาตรฐานโดยอัตโนมัติ เช่น ที่อยู่ IP และ user-agent ของเบราว์เซอร์ของคุณ เพื่อให้บริการหน้าเว็บและปกป้องบริการ การประมวลผลดังกล่าวอยู่ภายใต้ คำชี้แจงความเป็นส่วนตัวของ GitHub เราไม่มีสิทธิ์เข้าถึงข้อมูลการวิเคราะห์ที่ระบุตัวบุคคลเกี่ยวกับการเข้าชมของคุณ

7. บริการของบุคคลที่สาม

แอปนี้ไม่ได้ฝังบริการโฆษณา การวิเคราะห์ข้อมูล หรือการติดตามของบุคคลที่สามใด ๆ มันสร้างขึ้นบนไลบรารีซอฟต์แวร์โอเพนซอร์ส (เช่น SwiftNIO และ swift-crypto และ swift-certificates ของ Apple) ที่ทำงานบนเครื่องในฐานะส่วนหนึ่งของแอป และไม่ส่งข้อมูลของคุณไปยังที่ใด

8. ความปลอดภัย

เนื่องจากข้อมูลที่ดักจับได้และกุญแจส่วนตัวของ CA ของคุณอยู่บนอุปกรณ์ของคุณเท่านั้น ความปลอดภัยของข้อมูลเหล่านั้นจึงผูกอยู่กับความปลอดภัยของอุปกรณ์ของคุณ เราขอแนะนำให้คุณใช้การเข้ารหัสทั้งดิสก์ รหัสผ่านอุปกรณ์ที่รัดกุม และการป้องกันบัญชีของระบบปฏิบัติการ โปรดจำไว้ว่าทราฟฟิกที่ถอดรหัสแล้วและ CA รากที่เชื่อถือนั้นมีความละเอียดอ่อน จงปฏิบัติต่อไฟล์ที่ส่งออก (เช่น การส่งออก HAR หรือ CA ที่ส่งออก) อย่างเหมาะสม เนื่องจากไฟล์เหล่านั้นอาจมีข้อมูลรับรอง/บัญชีและรหัส

9. ความรับผิดชอบของคุณ

HTTrail เป็นเครื่องมือดักจับที่ทรงพลังซึ่งมีไว้สำหรับการแก้จุดบกพร่องของทราฟฟิกที่คุณได้รับอนุญาตให้ตรวจสอบ — แอปของคุณเอง อุปกรณ์ของคุณเอง และระบบที่คุณได้รับอนุญาตให้ทดสอบ คุณมีความรับผิดชอบในการใช้มันอย่างถูกกฎหมายและเฉพาะกับทราฟฟิกที่คุณมีสิทธิ์เข้าถึงเท่านั้น โปรดดู ข้อกำหนดการใช้งาน สำหรับข้อกำหนดการใช้งานที่ยอมรับได้ฉบับเต็ม

10. ความเป็นส่วนตัวของเด็ก

HTTrail เป็นเครื่องมือสำหรับนักพัฒนาที่มีไว้สำหรับมืออาชีพด้านซอฟต์แวร์ และไม่ได้มุ่งเป้าไปที่เด็ก เราไม่เก็บรวบรวมข้อมูลจากเด็กโดยเจตนา — อันที่จริง เราไม่เก็บรวบรวมข้อมูลจากใครเลยโดยเจตนา

11. ผู้ใช้ระหว่างประเทศ

เนื่องจากแอปประมวลผลข้อมูลเฉพาะบนอุปกรณ์ของคุณเองและเราไม่เก็บรวบรวมสิ่งใดเลย จึงไม่มีข้อมูลส่วนบุคคลใดที่ถูกถ่ายโอนข้ามพรมแดนโดยเรา เมื่อคุณใช้แอปเพื่อติดต่อเซิร์ฟเวอร์ของบุคคลที่สาม คุณเป็นผู้กำหนดว่าทราฟฟิกนั้นจะไปที่ใด

12. การเปลี่ยนแปลงนโยบายนี้

เราอาจปรับปรุงนโยบายความเป็นส่วนตัวนี้เป็นครั้งคราว — เช่น เพื่อสะท้อนคุณสมบัติใหม่ ๆ เมื่อเราทำเช่นนั้น เราจะแก้ไขวันที่ "อัปเดตล่าสุด" ด้านบน การเปลี่ยนแปลงที่มีนัยสำคัญจะถูกแสดงไว้บนหน้านี้ การที่คุณใช้แอปต่อไปหลังจากการปรับปรุงถือเป็นการยอมรับนโยบายฉบับแก้ไข
